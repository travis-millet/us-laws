# Supreme Court — Landmark Cases

This directory contains markdown files for the most significant Supreme Court decisions in US history.

## About This Collection

The Supreme Court is not a legislature — it doesn't write laws. But it interprets them, and those interpretations have the force of law. The cases here represent moments when the Court's interpretation fundamentally changed what the Constitution means in practice.

## The Cases

Each case file includes:
- Citation and year
- Which constitutional provisions are interpreted
- What the ruling held
- Why it matters constitutionally

- [Plessy v. Ferguson (1896)](landmark-cases/1896-plessy-v-ferguson.md) — 'Separate but equal' (overruled 1954)

### Progressive Era and Lochner Era (1900-1937)

- [Gitlow v. New York (1925)](landmark-cases/1925-gitlow-v-new-york.md) — First incorporation case
- [Buck v. Bell (1927)](landmark-cases/1927-buck-v-bell.md) — Forced sterilization (never formally overruled)
